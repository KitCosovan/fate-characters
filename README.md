# Fate Characters

Fate Characters — это веб-приложение для управления персонажами, NPC и кампаниями в играх по системе Fate (Fate Core, Fate Accelerated, Книга Пепла). 
Приложение упрощает ведение листов персонажей и организацию игр как для Мастеров (GM), так и для Игроков.

## 🌟 Возможности приложения

* **Регистрация и авторизация**: Безопасный вход в систему для хранения данных в облаке.
* **Поддержка Fate Core**: Классический лист с 18 навыками (пирамида до +4), 5 аспектами и 5 трюками.
* **Поддержка Fate Accelerated**: Упрощенный лист с 6 подходами вместо навыков.
* **Поддержка "Книги Пепла"**: Кастомный лист (20 навыков, шрамы, 6 слотов экипировки).
* **Создание кампаний**: Объединение игроков и NPC в рамках одной истории.
* **Роли в кампании**: Выбор роли Мастера (GM) или Игрока при присоединении к кампании.
* **Управление NPC**: Эксклюзивный доступ Мастера к созданию и ведению неигровых персонажей.
* **Мультиязычность**: Полный перевод интерфейса на русский (RU) и английский (EN) языки.
* **Тёмная и светлая темы**: Переключение оформления для комфортной игры в любых условиях.
* **Встроенная энциклопедия**: Быстрый доступ к терминам и правилам системы без выхода из приложения.
* **Справочник правил**: Встроенная книга правил для оперативного разрешения игровых ситуаций.
* **Раздел FAQ**: Ответы на популярные вопросы по системе и использованию приложения.
* **Удобный поиск**: Мгновенная фильтрация персонажей, NPC и кампаний по названию.
* **Фильтры по системам**: Сортировка созданных персонажей по выбранной системе правил.
* **Кастомизация**: Настройка цвета карточек кампаний для визуального разделения.

## 🚀 Установка локально

1. Клонируйте репозиторий: `git clone <url>`
2. Установите зависимости: `npm install`
3. Создайте `.env` и добавьте `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`.
4. Запустите проект: `npm run dev`
5. Откройте `http://localhost:5173`
